
# UIA Economy

Unity's Internal Agency economy system for school.

## Features
- Vox currency
- Transfers
- Snack marketplace
- Jobs
- Gambling
- Investments
- Banking
- Transaction logs
- UIA government theme

## Deploy

### GitHub
1. Create a GitHub repo
2. Upload all files
3. Push repo

### Render
1. Go to https://render.com
2. Create Web Service
3. Connect GitHub repo
4. Build command:
   npm install
5. Start command:
   npm start

### Local Run
npm install
npm start

Then open:
http://localhost:3000
